The full stack developer definition is “someone who can work on both the back-end and front-end of systems.” This means that they can develop fully fledged platforms (with databases, servers and clients) which don't need other applications to function.

MERN is an acronym used to describe a specific set of JavaScript based technologies that are used in the web application development process. It is designed with an idea to make the development process as smooth as possible.

MERN stands for MongoDB, Express, React, Node, after the four key technologies that make up the stack. MongoDB - document database. Express(.js) - Node.js web framework. React(.js) - a client-side JavaScript framework.
